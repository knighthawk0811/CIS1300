Homework 04 “Make a Rainbow”

Instructions:

Create a webpage to include the following:


  • Create a “.html” page

    • a home page file named “index.html”

  • Include the following

    • BODY tag

      • background → a dark blue color

    • DIV tags for the layout

      • .page

        • 80% width

        • max width 900px

        • min width 400px

        • center aligned content (not text)

        • background → a purple

      • .header

        • 100% width

        • background → a light blue

      • .content

        • 100% width

        • background → a yellow

        • contain the following:

          • .sidebar

            • 20% width

            • on the right side

            • 1px border on the left side colored black

            • background → a green

            • padding →  1rem left and right only

            • auto scrollable text (will naturally fit in the box)

          • .main

            • 80% width

            • on the left side

            • background → a red

            • padding →  1rem

      • .footer

        • 100% width

        • text is centered

        • background → same as header

  • Inside the main area place 3 paragraphs of Lorem Ipsum

  • inside the sidebar place 2 paragraphs of Lorem Ipsum

  • inside the header place a title

  • inside the footer place a copyright line

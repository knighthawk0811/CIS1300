Homework 04 “Make a Rainbow”

Instructions:

Create a webpage to include the following:


  • Create a “.html” page

    • a home page file named “index.html”

  • Include the following

    • A table with a width of 80%

      • 7 rows

      • 2 columns

      • type the word “RAINBOW” in every cell

  • Link the provided external CSS file

    • Dont change the file name

  • Attach the IDs for each row from the CSS file to each row in your table

    • in numerical order (you will see a rainbow if done correctly)

  • Attach the proper CLASS to every “td” element

    • The left class belongs on the left

    • The right class belongs on the right
